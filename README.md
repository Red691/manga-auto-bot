# Manga Auto Bot

## Features
- Auto channel creation (userbot)
- Scrapes manga/manhwa/manhua sites (static/dynamic/JS)
- PDF chapter generation with custom naming
- Thumbnail from manga cover
- Minimal button UI (Telegram)
- Multi-channel support
- Automatic updates for new chapters
- Admin editing for channel info/images
- GitHub integration for repo status

## Setup
1. Clone repo, install requirements:  
   `pip install -r requirements.txt`
2. Fill in `config.py` with your API_ID, API_HASH, BOT_TOKEN, and admin IDs.
3. Run `python bot.py` and `python userbot.py` (for channel creation).
4. Add your site via bot button or command.
